# Aufgabe 16

Ändern Sie das Programm aus *Aufgabe 14/15* so ab, dass eine seperate Ausgabe erfolgt wenn die Lebenspunkte auf 20 oder
darunter fallen. Darüberhinaus dürfen die Lebenspunkte nie unter 0 fallen.

```
Eugene hat 100 Lebenspunkte.
Eugene hat noch 93 Lebenspunkte übrig.
Eugene hat noch 86 Lebenspunkte übrig.
...
Eugene hat noch 23 Lebenspunkte übrig.
Eugene hat nur noch 16 Lebenspunkte übrig!
Eugene hat nur noch 9 Lebenspunkte übrig!
Eugene hat nur noch 2 Lebenspunkte übrig!
Eugene hat nur noch 0 Lebenspunkte übrig!
Eugene ist kampfunfähig.
```
