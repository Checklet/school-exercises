# Aufgabe 14

Erstellen Sie ein Programm, bei der die Lebenspunkte – startend von 100 – vermindert werden, bis es 0 erreicht.

```
Eugene hat 100 Lebenspunkte.
Eugene hat noch 80 Lebenspunkte übrig.
Eugene hat noch 60 Lebenspunkte übrig.
Eugene hat noch 40 Lebenspunkte übrig.
Eugene hat noch 20 Lebenspunkte übrig.
Eugene hat noch 0 Lebenspunkte übrig.
```
