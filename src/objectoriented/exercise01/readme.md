# Aufgabe 1

Erstellen Sie eine Klasse `Train`.

# Aufgabe 2

Geben Sie `Train` ein Attribut `model`, worin der Modellname gespeichert werden soll.

# Aufgabe 3

Schreiben Sie für `Train` ein Konstruktor, mit dem der Modellname überwiesen und dem Attribut zugewiesen werden kann.

# Aufgabe 4

Schreiben Sie für `Train` eine Methode, mit der der Modellname zurückgegeben wird.

# Aufgabe 5

Erstellen Sie eine Klasse `Railway`.

# Aufgabe 6

Geben Sie `Railway` ein Attribut des Typs `Train`.

# Aufgabe 7

Schreiben Sie für `Railway` eine Methode, mit der der Eisenbahnstrecke ein Zug zugewiesen werden kann.

# Aufgabe 8

Schreiben Sie für `Railway` eine Methode, mit der der aktuelle Zug von der Eisenbahnstrecke entkoppelt wird (*weisen sie dem Attribut `null` zu*).

# Aufgabe 9

Schreiben Sie für `Railway` eine Methode, die zurückgibt, ob sie durch ein Zug bereits besetzt ist oder nicht.

# Aufgabe 10

Ergänzen Sie für die Methode von *Aufgabe 7* den Aufruf zu der Methode von *Aufgabe 9*, um sicherzustellen dass ein Zug nur dann zugewiesen wird, wenn die Eisenbahnstrecke nicht belegt ist.
